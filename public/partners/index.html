API Overview

The public API is namespaced under /api/public/v1 to guarantee backwards compatibility. Responses include structured metadata, strong auditing guarantees, and are protected with rate limiting by default. Authenticate requests with your assigned X-API-Key header.

Health Check GET /api/public/v1/health Monitor service uptime and Supabase RLS enforcement.
Triage Reports POST /api/public/v1/triage-reports Generate AI-assisted triage analysis with automatic persistence.
Knowledge Base Articles POST /api/public/v1/knowledge-base/articles Create ready-to-publish documentation from live ticket context.

TypeScript SDK

Install the officially supported SDK to integrate faster with built-in typing, error handling, and fetch abstraction compatible with Node.js and modern browsers.

Install npm install @int/triage-sdk
Usage 
import { IntSmartTriageClient } from '@int/triage-sdk';

const client = new IntSmartTriageClient({
  baseUrl: 'https://api.example.com',
  apiKey: process.env.INT_PARTNER_KEY
});

const triage = await client.createTriageReport({
  customerName: 'Casey',
  ticketSubject: 'Sync issue',
  issueDescription: 'Mobile app sync stalled',
  customerTone: 'frustrated',
  csrAgent: 'CSR-007'
});
